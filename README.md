# Projet GNS3

## Comment le lancer

1) Cloner le github sur votre machine :

``` bash
git clone https://github.com/ThibaudLAMBERT/Script_routing_projects.git
```

2) Deplacez vous dans le dossier :

``` bash
cd Script_routing_projects
```
